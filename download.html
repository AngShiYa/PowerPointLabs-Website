Getting Started

PowerPointLabs is a plug-in that works with Microsoft PowerPoint 2010 and above. There're just a few easy steps you need to follow to get started.

1. Download

Register for free to get the PowerPointLabs installation bundle here. We'll send you an email with the download link after registration.

2. Installation

Extract the files in the downloaded bundle to a convenient location, then double-click setup.bat to begin installation.

Note: If you're using Windows Vista or above, you'll see the following prompt:

Click the Yes button to allow the installation process to continue.

You should see the following window:

Click the Install button to continue. Once installation has completed, the following screen should appear.

Click Close to exit the installer.

Note: After completing the above steps, the following window might appear.

This warning can be safely ignored by clicking either This program installed correctly or Cancel.

To verify that installation was successful, just launch PowerPoint and look for the PowerPointLabs item on the Ribbon.

If you had PowerPoint open during the installation process, you'll have to close it and start it again before the Ribbon tab appears.

All Set?

The installation bundle comes with a tutorial which guides you through using PowerPointLabs' features; just open the PowerPointLabs Quick Tutorial.pptx file, and follow the steps there.

If you need more help, check out our documentation to get a detailed walkthrough of how to put PowerPointLabs to work.

Uninstalling

To uninstall PowerPointLabs, just follow the instructions here, and look for PowerPointLabs in the list of programs.
